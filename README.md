# 英语句子练习系统

一个基于1700条英语句子的在线练习网站。

## 功能
- 练习模式：随机显示中文，输入英文，提交后显示正确答案
- 统计功能：做题数、正确率、正确/错误数、每句出现次数
- 句库浏览：查看全部1700条句子，支持筛选和搜索

## 数据
- 商务英语：1200条（汇报、跨部门沟通、职场生活、面试求职、书面表达、工作岗位职责）
- 日常口语：500条（社交、购物、出行、饮食、家庭情感）

数据保存在浏览器本地存储中。
